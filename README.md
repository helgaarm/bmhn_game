# Bygg med Helsenorge - Game

A planned browser-based cooperative 3D fantasy learning experience built around the nine-step "Bygg med Helsenorge" actor journey.

## Status

Repository setup and architecture discovery. Game implementation has not started.

## Public repository policy

Only reviewed, anonymized, non-sensitive material may be committed. Internal source reports, personal contact information, operational findings, credentials, and real citizen data must remain outside this public repository.
